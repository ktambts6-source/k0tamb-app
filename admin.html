📖 Как работает Ranking
• Создайте команды и установите им начальный RP
• Команды автоматически сортируются по RP (от большего к меньшему)
• При победе команда получит RP, при поражении потеряет
• Изменение RP зависит от позиции команды (топ команды теряют больше)
• Данные синхронизируются в облако и доступны на всех устройствах

🌐 Облачное хранилище статистики игроков (JSONBin.io)

Синхронизация глобальной статистики через облачное хранилище jsonbin.io для резервного копирования и доступа с разных устройств.
Заполните поля Access Key и Bin ID, затем нажмите "Сохранить ключи". Ключи хранятся локально в браузере.

📖 Инструкция по настройке JSONBin
1. Зарегистрируйтесь на jsonbin.io и создайте бесплатный аккаунт
2. Перейдите в раздел API Keys или Access Keys и создайте новый ключ доступа
3. Нажмите + Create Bin и создайте хранилище с содержимым {"globalPlayerStats":{}}, затем нажмите Create
4. Найдите Bin ID — это строка в URL вашего хранилища (после последнего слеша)
5. Введите ключ и Bin ID в поля выше и нажмите Сохранить ключи доступа к облаку

🤖 AI-тренер и аналитика матчей (DeepSeek API)

API для интеграции с искусственным интеллектом DeepSeek для генерации тактических коллов и анализа матчей.
Тактические паузы / анализ итогов матча будут работать только при наличии API ключа.
Получите ключ на platform.deepseek.com в разделе API Keys → Create API Key.

🔗 API симуляции матчей

Сгенерируйте прямую ссылку для открытия симулятора с заданными картами, командами и форматом. Вставьте её в адресную строку браузера — матч будет автоматически симулирован. Можно включить автоскачивание фото результата.
